USB에 분석환경을 설치하기 위해서는 먼저 WinPython을 다운로드 해서 USB에 위치시키다음 
배치파일 Setup_Environment_3.10.bat과  Setup_Environment_Auto.bat를 사용할 수 있습니다. 
Setup_Environment_3.10.bat은 파이썬 3.10이 설치된 것으로 필자가 만든 환경과 일치합니다. 

만약 자신이 최신의 파이썬을 설치하고자 한다면 Setup_Environment_Auto.bat를 사용하면 됩니다. 
다만 이때에는 필자가 설치한 라이브러리 역시 상위버전이 사용될 수 있기 때문에 
기존의 코딩과 버전이 일치하지 않을 수 있습니다.

하지만 이런 문제는 Google Gemini나 Claude를 이용하면 쉽게 해결됩니다.
